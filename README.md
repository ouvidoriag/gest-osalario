# Dashboard Financeiro para Casal

Um dashboard completo para controle financeiro de casais, com gerenciamento de entradas, saídas, categorias, tags e projeção mensal.

## Funcionalidades

- ✅ **Entradas e Saídas**: Controle completo de receitas e despesas
- ✅ **Categorias**: Organize transações por categorias personalizadas
- ✅ **Tags**: Adicione tags para melhor organização
- ✅ **Projeção Mensal**: Visualize projeções financeiras por mês
- ✅ **Gráficos**: Visualizações interativas com gráficos de pizza e barras
- ✅ **Parcelas**: Controle de transações parceladas
- ✅ **Banco de Dados SQLite**: Usa SQL.js (SQLite no navegador) - simples, leve e poderoso
- ✅ **Migração Automática**: Migra automaticamente dados do localStorage para SQLite

## Tecnologias

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Recharts (gráficos)
- Lucide React (ícones)
- date-fns (manipulação de datas)
- **SQL.js** (SQLite no navegador) - Banco de dados leve e simples

## Instalação

```bash
npm install
```

## Executar

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Estrutura

- `src/components/` - Componentes React
- `src/context/` - Context API para gerenciamento de estado
- `src/types/` - Definições TypeScript
- `src/utils/` - Funções utilitárias
- `src/data/` - Dados iniciais

## Dados Iniciais

O dashboard já vem com dados de exemplo baseados na lista de dezembro fornecida, incluindo todas as despesas listadas.
